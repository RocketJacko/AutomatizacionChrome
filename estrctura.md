## Estructura del Proyecto

```plaintext
.
├── Configuracion.js               # Archivo de configuración principal
├── Instancias.js                 # Gestión de instancias de Chrome
├── Main.js                       # Archivo principal de la aplicación
├── Perfiles.js                   # Gestión de perfiles de usuario
├── profileManager.js             # Clase principal que gestiona los perfiles
├── Puertos.js                    # Gestión de puertos
├── package.json                  # Dependencias y configuración del proyecto
├── .vscode/                      # Configuración específica de Visual Studio Code
│   └── settings.json             # Configuración del entorno de desarrollo
├── config/                       # Archivos de configuración adicionales
├── data/                         # Archivos de datos
├── Documentacion/                # Documentación del proyecto
│   └── Dependencias Instaladas.txt # Lista de dependencias instaladas
├── Respaldos/                    # Directorio de respaldos y documentación
│   └── Documentacion.md          # Archivo de documentación oficial
└── test/                         # Pruebas del proyecto
````
